# 🏋️ Fitness Tracker DBMS Project

A web-based Fitness Tracker application developed using **Python Flask**, designed to help users manage workouts, diet plans, fitness goals, trainers, activities, and badges in one platform.

This project demonstrates the implementation of a **Database Management System (DBMS)** with a user-friendly interface and multiple fitness-related modules.

---

# 🚀 Features

* 🔐 User Login System
* 🏋️ Workout Management
* 🍎 Diet Tracking
* 🎯 Goal Setting
* 🏃 Activity Monitoring
* 🧑‍🏫 Trainer Management
* 🤝 Friend Connectivity
* 🏅 Badge & Achievement System
* 📊 Dashboard Overview
* 💾 Database Integration

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/fitness-tracker-dbms.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd fitness-tracker-dbms
```

## 3️⃣ Install Required Packages

```bash
pip install flask
```

## 4️⃣ Run the Application

```bash
python app.py
```

## 5️⃣ Open in Browser

```bash
http://127.0.0.1:5000/
```
